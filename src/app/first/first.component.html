About Chromispos Angular.

This is just simple apps for modern front end chromipos point of sale software, Develope using Angular framework with integration on chromispos mysql databased and bootstrap for CSS UI.
You can modify apps sourcecode and clouds this apps if you needed, just run ng build and upload file on your host,dont forget to connected with your clouds chromispos database.
If you need another web apps you can call us for web apps services, present by https://laravelapp.sourceforge.io.
